<h1 style="color:cyan;"> GIT </h1>

CONCEPTS:

- Repositories: Folders for files, managed by git.
- Commits: Set of changes in a repository.
- Branches
- Merging
- Tags

COMMANDS:

- git init
- git add \<file>
- git pull [origin xxxx]
- git push [origin xxxx]
- git branch
- git branch -d 
- git checkout | git switch
- git checkout -b \<branch name>
- git log
- git merge
- git rebase
- git diff
- git tag
- git stash
- git reset
- HEAD
